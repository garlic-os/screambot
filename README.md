# Screambot
A Discord bot that screams. That's it.

Screams when:
- Pinged
- Someone else screams
- Someone says something (sometimes)

## Requirements
- `boost`
- `libdpp` — install from https://dpp.dev/

## Usage
2. `$ make`
1. Make a copy of `config.example.json` named `config.json` and fill it in
3. `$ ./screambot`
